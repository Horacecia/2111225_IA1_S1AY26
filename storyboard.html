Desktop Layout Design

The desktop layout is designed for screens with a minimum width of 950px. It includes a header with the logo, welcome message, and navigation bar. The main content area displays the page heading, profile image, and related content in a clean and organized layout. A footer is placed at the bottom with copyright information and a Back to Top link.

HTML groups shown in the desktop wireframe: <header>, <nav>, <main>, <section>, <article>, and <footer>.

Mobile Layout Design

The mobile layout is designed for screens with a minimum width of 600px. The layout stacks content vertically so it is easier to read on smaller screens. The logo and menu are placed at the top, followed by the welcome message, profile image, content section, and footer.

The mobile design uses the same HTML groups as the desktop version, but the content is arranged in a single-column layout for better readability.

3. Colour Palette

Figure 3: Colour Palette Used for the Website

The colour palette was carefully selected to create a modern, professional, and elegant appearance. Black and dark grey provide a strong foundation, while gold adds emphasis and complements the HP logo. Cream and white create a clean background that improves readability.

Primary Black: #111111 — RGB (17, 17, 17)

Dark Gray: #1F1F1F — RGB (31, 31, 31)

Gold Accent: #D4AF37 — RGB (212, 175, 55)

Cream Background: #F8F4E8 — RGB (248, 244, 232)

White: #FFFFFF — RGB (255, 255, 255)

Text Colour: #222222 — RGB (34, 34, 34)
