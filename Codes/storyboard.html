Storyboard

1. Desktop Layout Design (Minimum Width: 950px)

Figure 1: Desktop Wireframe Layout

The desktop layout was designed for screens with a minimum width of 950px. The page begins with a header containing the HP logo, website title, welcome message, and navigation bar. The main section displays the page heading followed by the content arranged neatly inside sections and articles. The footer is positioned at the bottom of the page and contains copyright information together with a Back to Top link.

HTML groups represented in the wireframe include <header>, <nav>, <main>, <section>, <article>, and <footer>.

2. Mobile Layout Design (Minimum Width: 600px)

Figure 2: Mobile Wireframe Layout

The mobile layout was designed for screens with a minimum width of 600px. The layout is displayed in a single-column format to improve readability on smaller devices. The navigation links are stacked vertically while the content flows naturally from the header to the footer for a better mobile experience.

The same HTML groups are used in the mobile version, but the content is reorganized to fit smaller screen sizes while maintaining the same design and functionality.

3. Colour Palette

Figure 3: Colour Palette Used for the Website

The colour palette was carefully selected to create a modern, professional, and elegant appearance. Black and dark grey provide a strong foundation, while gold adds emphasis and complements the HP logo. Cream and white create a clean background that improves readability.

Primary Black: #111111 — RGB (17, 17, 17)

Dark Gray: #1F1F1F — RGB (31, 31, 31)

Gold Accent: #D4AF37 — RGB (212, 175, 55)

Cream Background: #F8F4E8 — RGB (248, 244, 232)

White: #FFFFFF — RGB (255, 255, 255)

Text Colour: #222222 — RGB (34, 34, 34)
